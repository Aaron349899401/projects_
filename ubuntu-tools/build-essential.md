# build-essential

## What it is for

`build-essential` installs core C/C++ build tools (compiler, linker, make).

## Install

```bash
sudo apt install -y build-essential
```

## Why you need it

- Native Node/Python package builds
- Compiling many open-source tools
- Local C/C++ development

## Verify

```bash
gcc --version
make --version
```
