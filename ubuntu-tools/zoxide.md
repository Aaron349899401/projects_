# zoxide

## What it is for

`zoxide` is a smarter `cd` that remembers and ranks your frequent directories.

## Setup

```bash
eval "$(zoxide init bash)"
```

Add that line to `~/.bashrc`.

## Common examples

```bash
z neural-net-visualizer
z projects
zi
```

## Why use it

- Faster navigation than manual long paths
- Learns over time
- Great with many repos
