# zip and unzip

## What they are for

- `zip`: create compressed archives
- `unzip`: extract archive contents

## Common examples

```bash
zip -r project.zip my-folder
unzip project.zip
unzip project.zip -d extracted/
```

## Practical use

- Package deliverables
- Share code snapshots
- Handle release downloads

## Tip

Use `-r` for folders and `-d` to control extraction location.
