# Ubuntu Terminal Tools Guide

This folder contains one file per tool from the recommended Ubuntu terminal stack.

How to use this folder:

- Start with `git.md`, `ripgrep-rg.md`, `fzf.md`, and `tmux.md` for the biggest productivity gains.
- Use `install-packages.md` to install the core tools quickly.
- Read each tool file for:
  - what the tool is for
  - common command patterns
  - practical examples
  - mistakes to avoid

Suggested learning order:

1. `git.md`
2. `ripgrep-rg.md`
3. `fd-fdfind.md`
4. `fzf.md`
5. `zoxide.md`
6. `tmux.md`
7. `jq.md`
8. `gh-github-cli.md`
