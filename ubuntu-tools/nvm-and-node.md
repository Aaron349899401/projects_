# nvm and Node.js

## What they are for

- `nvm` manages multiple Node versions safely.
- Node.js runs JavaScript tooling and apps.

## Common commands

```bash
nvm install --lts
nvm use --lts
nvm ls
node -v
npm -v
```

## Why use nvm

- Avoids system-wide Node conflicts
- Easy version switching per project
- Cleaner global package management
