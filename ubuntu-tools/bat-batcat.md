# bat (batcat on Ubuntu)

## What it is for

`bat` is a better `cat` with syntax highlighting and line numbers.

Ubuntu command name is often `batcat`.

## Common examples

```bash
batcat README.md
batcat -n script.py
batcat --paging=never file.txt
```

## Why use it

- Easier file reading
- Better visual scanning
- Great for logs and configs

## Tip

Add alias:

```bash
alias cat='batcat'
```
