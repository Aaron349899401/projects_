# neovim

## What it is for

`neovim` is a powerful modal terminal editor.

## Quick start

```bash
nvim file.txt
```

Core motions:

- `i` insert mode
- `Esc` normal mode
- `:w` save
- `:q` quit
- `:wq` save and quit

## Why use it

- Very fast keyboard-driven editing
- Highly customizable for coding
