# pnpm

## What it is for

`pnpm` is a fast, disk-efficient JavaScript package manager.

## Install

```bash
npm i -g pnpm
```

## Common commands

```bash
pnpm install
pnpm add axios
pnpm add -D typescript
pnpm run dev
```

## Why use it

- Faster installs in many projects
- Better disk usage than npm/yarn in large environments
