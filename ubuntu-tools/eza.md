# eza

## What it is for

`eza` is a modern replacement for `ls` with better defaults and Git-aware views.

## Common examples

```bash
eza
eza -la
eza -la --group-directories-first
eza -T
```

## Why use it

- Cleaner listing output
- Better file metadata display
- Optional tree output

## Tip

Popular alias:

```bash
alias ls='eza -la --group-directories-first'
```
