# gh (GitHub CLI)

## What it is for

`gh` lets you manage repos, issues, PRs, reviews, and CI from terminal.

## Setup

```bash
gh auth login
gh repo view
```

## Daily commands

```bash
gh pr create
gh pr view --web
gh pr checkout 123
gh issue list
gh run list
```

## Why it helps

- Faster PR workflow than browser-only flow.
- Easy automation in scripts and CI.

## Tip

Pair `gh` with `git` aliases for a fast review loop.
