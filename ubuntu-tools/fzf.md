# fzf

## What it is for

`fzf` is an interactive fuzzy finder for files, command history, and custom lists.

## Common examples

```bash
fzf
history | fzf
git branch | fzf
```

## Powerful pattern

Combine with other tools:

```bash
fdfind | fzf
rg "TODO" | fzf
```

## Why use it

- Finds things quickly with partial matches
- Works inside many shell workflows

## Tip

Enable keybindings/completion for best experience.
