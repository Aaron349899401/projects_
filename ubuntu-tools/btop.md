# btop

## What it is for

`btop` is a modern resource monitor for CPU, memory, disks, and network.

## Usage

```bash
btop
```

## Why use it

- Better visuals than many classic monitors
- Good at spotting spikes and bottlenecks quickly
- Includes process list and system stats together

## Tip

Use it when running builds, servers, or ML jobs to monitor pressure.
